# 🤖 Friday Voice Assistant (Python AI Project)

🚀 This is my first AI project built using Python.  
Friday is a voice assistant that can listen, speak, and respond using AI.

It works like a mini **Jarvis / Friday assistant** and can perform voice commands, open websites, generate AI replies, and read news.

---

## ✨ Features

- 🎙 Wake word detection (Friday)
- 🗣 Voice command recognition
- 🌐 Open websites (Google, YouTube, LinkedIn, etc.)
- 🤖 AI chat using Ollama (Llama3)
- 📰 News headline generation
- 🔊 Text to Speech output
- ⚡ Real-time voice interaction

---

## 🛠 Technologies Used

- 🐍 Python
- 🎤 SpeechRecognition
- 🔊 gTTS (Google Text to Speech)
- 🎮 Pygame
- 🤖 Ollama (LLM - Llama3)
- 🗣 pyttsx3
- 🌐 Webbrowser
- 📡 Requests

---

## 📌 Project Description

This project is a simple AI Voice Assistant built using Python.  
It listens for the wake word **"Friday"**, takes voice commands, and performs actions like opening websites, generating AI responses, and reading news using a local LLM.

This project helped me understand:
- Voice recognition
- Text-to-speech
- AI integration
- Python automation
- Working with LLM (Ollama)
